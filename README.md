## About This Repository

This repository represents my early Python learning journey at IIT Jodhpur during the first semester of my first year. The programs were developed as part of the course Algorithmic Thinking and its Applications, where I was introduced to fundamental programming concepts and problem-solving techniques using Python. This collection reflects my initial hands-on experience with coding and serves as a record of the foundational skills I developed during the beginning of my undergraduate studies.
